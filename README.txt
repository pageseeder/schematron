=================================
WEBORGANIC - Schematron Validator
=================================

About this library
------------------

This library provides a simple Schematorn validator in Java.

Dependencies
------------

This library requires an XSLT2 processor (For example Saxon9)

Testing
-------

If you want to test the code, ensure that you add the following to the ANT classpath:
- Saxon (available in text/lib)
- Either a version of the jar or both the /src and /classes folders
